How to choose where files save

Files download using your browser settings. Enable 'Ask where to save each file before downloading' in your browser if you want to choose the save location.

  • Chrome: Settings > Downloads > Toggle "Ask where to save each file before downloading"
  • Firefox: Settings > General > Files and Applications > Select "Always ask you where to save files"
  • Safari: Preferences > General > File download location > Select "Ask for each download"
